# SetStyles
Google Chrome extension


## Use
styles will be applied to the specified url and all child path

1. `https://site.com/group`
applicable: <br>
`https://site.com/group123`
`https://site.com/group/**`
not applicable:
`https://site.com/friends`

2. `https://site.com`
applicable for all links
